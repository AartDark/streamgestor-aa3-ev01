Resumen del módulo

El sistema permite realizar operaciones de inserción, consulta, actualización y eliminación sobre los módulos principales del proyecto.

La estructura fue desarrollada con Spring Boot, Thymeleaf y base de datos H2 para facilitar la ejecución local.
