# StreamGestor - GA7-220501096-AA3-EV01

Proyecto web desarrollado como evidencia de codificación de módulos del software.

## Tecnologías

- Java
- Spring Boot
- Thymeleaf
- Spring Data JPA
- H2 Database
- Maven
- HTML, CSS y JavaScript

## Funcionalidades

- Inicio de sesión de administrador.
- Panel principal.
- CRUD de clientes.
- CRUD de categorías.
- CRUD de películas.
- CRUD de reservas.
- Validaciones básicas en formularios.
- Diseño web administrativo sin imágenes externas.

## Ejecución

1. Abrir el proyecto en NetBeans.
2. Ejecutar `StreamGestorApplication.java`.
3. Ingresar a: http://localhost:8082

El puerto fue configurado en 8082 para evitar conflictos con ejecuciones anteriores.
